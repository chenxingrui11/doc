        • 因为方便

        • ssh-keygen -t dsa

      • scp
        • scp .ssh/*.pub username@remote.host:~/

        • $ cd ~/.ssh $ cat ../identity.pub >> authorized_keys $ cat ../id_rsa.pub >> authorized_keys2 $ cat ../id_dsa.pub >> authorized_keys2
