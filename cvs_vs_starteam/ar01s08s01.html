8.1. 用同一套本地目录，对应于相应的 STARTEAM 和 CVS 工程

导出工程在 Starteam 中的文件和目录结构，并在 CVS 中创建新的工程“project”，并导入目录结构。 

# cd ~jiangxin/work/
# mkdir project
# cd project
# st config

# st cd 
# cvs import -m "create directory" project Johnson init
